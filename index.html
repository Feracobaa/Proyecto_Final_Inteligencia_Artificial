G y H. Despliegue y Uso de IA Generativa

G. Despliegue Cloud

Esta aplicación fue construida exclusivamente en el front-end (HTML/JS/CSS) sin necesidad de un backend. Esto permite alojarla y desplegarla fácilmente y sin costo en entornos cloud estáticos como GitHub Pages. Todo el procesamiento matemático pesado corre de manera distribuida en el navegador del cliente.

H. Herramientas de IA Generativa en el Desarrollo

Durante el desarrollo de esta plataforma se utilizó Google DeepMind Antigravity (Gemini) como pair programmer agentic.

  • ¿Qué fue generado por la IA?
    La estructura HTML semántica interactiva, los estilos CSS complejos "Glassmorphism", y toda la base algorítmica matemática en `app.js` (parseo CSV, normalización matemática, descenso del gradiente de Regresión Logística y cálculo de métricas matriciales).
  • ¿Qué fue corregido/ajustado?
    Se corrigieron problemas de diseño arquitectónico al pasar de una simulación lineal 2D a una herramienta de Machine Learning que maneja CSVs multivariables. Se ajustaron las fórmulas de normalización (Z-Score) esenciales para evitar que el Gradiente Descendiente diverja en features de escalas grandes.
  • ¿Qué fue comprendido por el estudiante?
    A través de este desarrollo y uso iterativo, el estudiante demuestra que comprende cómo funcionan intrínsecamente las librerías como Scikit-Learn. Comprende que una regresión logística no es una "caja negra", sino una combinación lineal seguida de una función sigmoide optimizada mediante cálculo de derivadas (Log-Loss). Entiende la importancia de no evaluar sobre los datos de entrenamiento para prevenir Data Leakage (fuga de datos).
